# Nano Banana Studio — Cowork Folder Instructions

## Description
Single-file HTML app for generating images using the Gemini API (Nano Banana 2
and Nano Banana Pro models). Includes bulk import via JSON arrays and JSZip
download of generated images. Currently at v1.0.

## Rules
- Single-file architecture — HTML + CSS + JS all in index.html
- Do not split into separate files unless Wayne explicitly asks
- British English in all UI text and comments
- Never hardcode API keys in the HTML file
- Bulk import uses stringified JSON prompts inside the prompt field

## Key Files
- `index.html` — the entire application
- `TODO.md` — planned improvements

## Output Folder
All output stays in this project folder.
